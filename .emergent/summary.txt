**Key Technical Concepts**
- **Backend**: FastAPI, Python, Pydantic, JWT pour l'authentification.
- **Frontend**: React, JavaScript, Shadcn/UI, Tailwind CSS, Recharts.
- **Database**: FerretDB (proxy compatible MongoDB) avec PostgreSQL comme base de données sous-jacente. L'ensemble fonctionne dans Docker.
- **Déploiement**: Géré par l'utilisateur sur un Raspberry Pi, utilisant Nginx comme reverse proxy et des services  pour le backend.

**key DB schema**
- Les modèles sont définis dans  via Pydantic.
- **users**: Stocke les informations des utilisateurs.
- **items**: Modèle générique pour l'inventaire, la liste de souhaits, etc., différencié par un champ . Contient des champs pour le nom, la description, les tags, les métadonnées personnalisées.

- **portfolio_assets**: Stocke les actifs du portefeuille avec quantité, prix d'achat, etc.
- **alerts**: Stocke les alertes de prix.
